# vault-sync
It enables you to effortlessly store IP addresses, URLs, usernames, and passwords, automatically verify their validity, and synchronize them to Vaultwarden.

## git usages

### create a new repository on the command line
```
echo "# vault-sync" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:nino-dufresne/vault-sync.git
git push -u origin main
```

### push an existing repository from the command line
```
git remote add origin git@github.com:nino-dufresne/vault-sync.git
git branch -M main
git push -u origin main
```
